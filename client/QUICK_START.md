# Patient Management System - Frontend

React 18 application with TypeScript for managing patient records, appointments, and consultations.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```

## Documentation

See [README.md](./README.md) for full documentation.

## Technology Stack

- React 18 + TypeScript (strict mode)
- Material-UI v9
- React Router v7
- Axios with interceptors
- Vitest + Testing Library
